Integrasi Google Sheets Storage

Gunakan Google Sheets sebagai penyimpanan cloud untuk kolaborasi multi-perangkat.

Masukkan URL Web App dari Google Apps Script yang terhubung dengan Spreadsheet Anda. Kosongkan untuk tetap menyimpan data secara lokal pada web browser ini.

Cara Menghubungkan ke Google Sheets:

  1. Buat sebuah Spreadsheet Baru di Google Sheets.
  2. Buka menu Ekstensi → Apps Script.
  3. Hapus semua kode bawaan, lalu salin dan tempel kode pemrograman di bawah ini ke dalamnya.
  4. Klik tombol Terapkan (Deploy) → Penerapan Baru (New Deployment).
  5. Pilih jenis Aplikasi Web (Web App).
  6. Ubah akses 'Siapa yang memiliki akses' menjadi Siapa saja (Anyone). Lalu jalankan deploy & berikan otorisasi.
  7. Salin URL Aplikasi Web (Web App URL) yang muncul dan tempel pada kolom di atas, lalu klik Simpan Perubahan Branding.
